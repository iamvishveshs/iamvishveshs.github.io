Features

  1. User Authentication: Secure login and registration system for both Admins and Users to manage their profiles and listings.

  2. Property Listings: Users can view detailed property descriptions, including amenities, pricing, location, and images.

  3. Advanced Search & Filters: Functionality to sort and filter properties based on:

    • Type: Apartment, House, PG, Hostel.

    • Status: For Rent.

    • Budget: Price range filtering.

  4. Admin Dashboard: A dedicated control panel for administrators to approve listings, manage users, and oversee platform activity.
